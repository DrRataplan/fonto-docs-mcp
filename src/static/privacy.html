Privacy Policy

Last updated: 2026-07-24

Fonto Docs MCP (fonto-docs.elliat.nl) is a server that fetches Fonto's public documentation and converts it to Markdown for AI tools. It does not require an account, does not use cookies, and does not run any analytics or advertising trackers.

What is collected

  • Request logs. Every MCP tool call and HTTP API request (/search, /page/, /catalog) is logged, including the tool name and the arguments you sent — for example, your search query, page slug, or keyword. These logs go to Google Cloud Logging because the server is hosted on Google Cloud Run.
  • IP addresses. Your IP address is held briefly in memory to apply a per-instance rate limit (60 requests/minute), then discarded — it is not written to the application logs above. Separately, Google Cloud Run automatically records standard infrastructure request logs (IP address, timestamp, path, status code, user agent) for every request to any Cloud Run service; this is a Google Cloud platform default, not something this application adds or can opt out of.
  • Nothing else. No names, emails, or other identifiers are collected, because none are ever submitted to the server.

Why it's collected

Request logs exist to debug errors and understand usage patterns (e.g. which tools are called, whether a lookup failed). Rate-limit IP tracking exists solely to blunt a single client from overwhelming the server.

Third parties

To answer a search or page-lookup request, the query, slug, or keyword you provide is forwarded live to documentation.fontoxml.com, Fonto's own public documentation site and search API, so this server can retrieve and relay the result. Fonto's servers see that request and are subject to Fonto's own privacy practices, not this one.

No data is sold, shared with advertisers, or used for profiling.

Retention

Logs are retained under Google Cloud Logging's standard retention settings for this project and are not reviewed except to debug reported problems. Nothing is stored in a database; the only in-process caches (page content, 10 minutes; page catalog, process lifetime) hold public documentation content, not user data.

Contact
